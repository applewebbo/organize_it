# Welcome to Organize It

**Organize It** is a powerful web application designed to help you plan and manage your trips with ease.

![Organize It](https://images.unsplash.com/photo-1488646953014-85cb44e25828?w=800&auto=format&fit=crop)
*Photo by Glenn Carstens-Peters on Unsplash*

## What is Organize It?

Organize It helps you create detailed travel itineraries by organizing all aspects of your trip in one place:

- **Trips** - Complete travel plans with automatic status updates
- **Days** - Auto-generated daily schedules
- **Stays** - Hotel and accommodation management
- **Experiences** - Activities, museums, tours, and attractions
- **Meals** - Restaurant reservations and dining plans
- **Transfers** - Flights, trains, and transportation

## Technology Stack

Organize It is built with modern web technologies:

- **Backend**: Django 6.0+, Python 3.14+
- **Frontend**: TailwindCSS, HTMX, Alpine.js
- **Database**: PostgreSQL (production), SQLite (development)
- **Maps**: Mapbox, Folium
- **Images**: Unsplash API integration
